# JS Custom Range Slider 🎚️

A custom range slider demo using HTML, CSS, and JavaScript. Demonstrates styling input range elements and handling user input.

## Features
- Styled range slider
- Live value display (if implemented)
- Beginner-friendly code

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
```
index.html — demo page
styles.css — slider styles
script.js — interaction logic
```

## Installation / How to run
1. Clone the repo.
2. Open `index.html` in a browser.
